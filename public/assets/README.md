# MASH Landing Page Assets

This directory contains all the assets used in the MASH landing page.

## Directory Structure

- `images/` - Logo, screenshots, and other images
- `icons/` - Icon files for various UI elements
- `fonts/` - Custom font files if needed
- `videos/` - Demo and promotional videos

## Video Files

Place the following video files in the `videos/` directory:

- `demo.mp4` - Main demo video (MP4 format)
- `demo.webm` - Main demo video (WebM format for broader compatibility)
- `overview.mp4` - System overview video
- `overview.webm` - System overview video (WebM format)
- `setup.mp4` - Installation and setup video
- `setup.webm` - Installation and setup video (WebM format)
- `mobile.mp4` - Mobile app demonstration video
- `mobile.webm` - Mobile app demonstration video (WebM format)
- `poster.jpg` - Video poster/thumbnail image

## Image Files

Recommended images to add:

- Logo files (PNG, SVG)
- Screenshot images
- Feature illustrations
- Background images

## Icons

Custom icon files can be placed here if needed, though the current implementation uses inline SVG icons.

## Usage

All assets should be optimized for web delivery:
- Videos: Compressed, under 10MB each
- Images: Optimized PNG/JPG/WebP format
- Icons: SVG format preferred for scalability
